Features

CSS-Magic gives you a huge collection of different CSS components,
which are looks nice, simple to use and easy to customize!

Sass

CSS-Magic is developed with Sass, the most powerful professional grade CSS extension language in the world

Components

CSS-Magic is a collection of small, separated components, that's mean you can use full library or just one element

Customize

CSS-Magic can be easily customized by your needs

Open Source

CSS-Magic is open source project and MIT licensed. It is absolutely free of charge and you can use, copy and publish it without any limitations.

Contribute

CSS-Magic is GitHub project and open for everyone to contribute. Just fork it on GitHub and send us a pull request.

Found a bug?

In case if you found a bug, please give us some feedback or open a new issue on GitHub.
